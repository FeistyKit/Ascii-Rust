# Ascii-Rust

A small program that I wrote to convert an image file to Ascii text.

## Usage
```bash
$ git clone https://www.github.com/FeistyKit/ascii-rust
$ cd ascii-rust
$ cargo build --release
$ target/release/ascii-ify ./socks.png -o socks.txt
```
